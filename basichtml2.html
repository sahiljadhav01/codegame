HTML Attributes and Elements Tutorial

HTML Attributes and Their Uses

HTML attributes provide additional information about HTML elements. Here are some commonly used attributes:

  • href: Specifies the URL of the link destination.
  • src: Specifies the URL of the image, audio, video, or script file.
  • alt: Provides alternative text for images if the image cannot be displayed.
  • title: Specifies extra information about an element, often displayed as a tooltip.
  • class: Assigns one or more class names to an element for styling with CSS.
  • id: Specifies a unique identifier for an element, used for scripting or styling purposes.
  • style: Sets inline CSS styles for an element.
  • target: Specifies where to open the linked document when clicked (e.g., "_blank" for a new tab).

HTML Elements and Their Uses

HTML provides various elements to structure content on a webpage. Here are some commonly used elements:

  • <h1> - <h6>: Heading elements for specifying different levels of headings.
  • <p>: Paragraph element for containing text paragraphs.
  • <a>: Anchor element for creating hyperlinks.
  • <img>: Image element for embedding images.
  • <ul>, <ol>, <li>: Elements for creating unordered and ordered lists with list items.
  • <div>: Division element for grouping content and applying styles.
  • <span>: Inline element for applying styles or scripting to a specific part of text.
  • <input>: Input element for accepting user input, such as text, checkboxes, or radio buttons.
  • <button>: Button element for creating clickable buttons.

Adding Audio and Video Files in HTML

To add audio and video files to a webpage, you can use the <audio> and <video> elements respectively.

Example of adding an audio file:

<audio controls>
    <source src="audio_file.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

Explanation:

  • <audio>: Defines an audio player.
  • controls: Adds playback controls (play, pause, volume) to the audio player.
  • <source>: Specifies the audio file to be played, with the src attribute containing the URL of the audio file and the type attribute indicating the MIME type of the audio file.
  • The content inside the <audio> element is fallback content displayed if the browser does not support the audio element.

Example of adding a video file:

<video controls width="400">
    <source src="video_file.mp4" type="video/mp4">
    Your browser does not support the video element.
</video>

Explanation:

  • <video>: Defines a video player.
  • controls: Adds playback controls (play, pause, volume) to the video player.
  • width: Sets the width of the video player (in pixels).
  • <source>: Specifies the video file to be played, with the src attribute containing the URL of the video file and the type attribute indicating the MIME type of the video file.
  • The content inside the <video> element is fallback content displayed if the browser does not support the video element.
