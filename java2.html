Java Basic Questions

1. What is the default value of byte variable in Java?

Answer: The default value of a byte variable in Java is 0.

2. What is the result of 7 % 3 in Java?

Answer: The result of 7 % 3 in Java is 1. The % operator returns the remainder of the division.

3. What is the use of 'final' keyword in Java?

Answer: In Java, the 'final' keyword is used to declare constants, restrict method overriding, and prevent inheritance.

4. What is the difference between '==' and '.equals()' method in Java?

Answer: The '==' operator is used to compare primitive data types and memory addresses of objects, whereas the '.equals()' method is used to compare the content or values of objects.

5. What is the purpose of the 'static' keyword in Java?

Answer: In Java, the 'static' keyword is used to create variables and methods that belong to the class rather than instances of the class. It allows these variables and methods to be accessed without creating an instance of the class.

6. What is the difference between 'String' and 'StringBuilder' in Java?

Answer: String objects are immutable in Java, meaning their values cannot be changed after they are created. StringBuilder, on the other hand, is mutable, allowing for the modification of its value without creating a new object.

7. What is the difference between an interface and an abstract class in Java?

Answer: An interface in Java can only contain abstract methods and cannot have method implementations, while an abstract class can have both abstract and non-abstract methods. Additionally, a class can implement multiple interfaces but can only extend one abstract class.

8. What is method overloading in Java?

Answer: Method overloading in Java allows a class to have multiple methods with the same name but with different parameters. The methods must have different parameter lists (number of parameters or data types of parameters).

9. What is method overriding in Java?

Answer: Method overriding in Java occurs when a subclass provides a specific implementation of a method that is already defined in its superclass. The method signature (name, return type, and parameter list) must be the same in both the superclass and subclass.

10. What is the difference between 'public', 'protected', 'private', and 'default' access modifiers in Java?

Answer: 'public' access modifier allows access from any other class, 'protected' allows access within the same package or by subclasses, 'private' allows access only within the same class, and 'default' (no modifier) allows access within the same package but not outside of it.

11. What is the purpose of the 'this' keyword in Java?

Answer: The 'this' keyword in Java refers to the current instance of the class. It can be used to differentiate between instance variables and parameters with the same name, to invoke one constructor from another in constructors chaining, and to pass the current object as a parameter to other methods.

12. What is the difference between 'ArrayList' and 'LinkedList' in Java?

Answer: ArrayList is implemented as a resizable array, while LinkedList is implemented as a doubly linked list. ArrayList provides fast iteration and random access but slower insertion and deletion of elements, whereas LinkedList provides fast insertion and deletion but slower iteration and random access.

13. What is the purpose of the 'super' keyword in Java?

Answer: The 'super' keyword in Java is used to refer to the immediate parent class object. It can be used to access methods and variables of the superclass, to invoke superclass constructor from subclass constructor, and to differentiate between superclass and subclass variables and methods.

14. What is the difference between '==', '.equals()', and 'compareTo()' methods in Java?

Answer: '==' is used to compare primitive data types and memory addresses of objects, '.equals()' method is used to compare the content or values of objects for equality, and 'compareTo()' method is used to compare objects for ordering, typically used for sorting objects in collections.

15. What is the purpose of the 'instanceof' operator in Java?

Answer: The 'instanceof' operator in Java is used to test whether an object is an instance of a particular class or interface. It returns true if the object is an instance of the specified type or its subclasses/interfaces, otherwise false.

16. What are constructors in Java?

Answer: Constructors in Java are special methods used for initializing objects. They have the same name as the class and do not have a return type. Constructors can be parameterized or default (no-args).

17. What is the purpose of the 'try', 'catch', 'finally', and 'throw' keywords in Java?

Answer: 'try' block is used to enclose the code that may throw exceptions, 'catch' block is used to handle the exception that occurred in the try block, 'finally' block is used to execute code regardless of whether an exception is thrown or not, and 'throw' keyword is used to manually throw an exception.

18. What are access modifiers in Java?

Answer: Access modifiers in Java are keywords used to set the accessibility of classes, variables, methods, and constructors. They include 'public', 'protected', 'private', and default (no modifier).

19. What is the purpose of the 'static' block in Java?

Answer: The 'static' block in Java is used for static initialization of a class. It is executed only once when the class is loaded into memory, before the execution of the main method or creation of objects of that class.

20. What is method hiding in Java?

Answer: Method hiding in Java occurs when a subclass defines a static method with the same signature as a static method in its superclass. It hides the superclass method rather than overriding it.
