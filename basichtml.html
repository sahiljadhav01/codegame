HTML Terminology Tutorial

HTML Terms

Here are some common terms used in HTML:

  • Tag: A tag is an element in HTML that defines the structure and content of a webpage. Tags are enclosed in angle brackets, e.g., <tag>.
  • Element: An element consists of a pair of tags along with the content enclosed within them. It represents a specific piece of content on a webpage.
  • Attribute: Attributes provide additional information about an element. They are specified within the opening tag of an element and are written as name-value pairs, e.g., attribute="value".
  • HTML Document: An HTML document is a file that contains HTML code. It typically consists of a series of nested elements that define the structure and content of a webpage.
  • DOCTYPE: The DOCTYPE declaration specifies the version of HTML used in the document and helps browsers to render the page correctly.
  • HTML5: HTML5 is the latest version of HTML, introducing new elements, attributes, and features for web development.
  • Semantic Elements: Semantic elements provide meaning to the content they enclose, making it easier for search engines and screen readers to interpret the page. Examples include <header>, <footer>, <nav>, and <article>.
  • Inline vs. Block Elements: Inline elements flow within the text and do not start on a new line, while block elements start on a new line and take up the full width available. Examples of inline elements include <span> and <a>, while examples of block elements include <div> and <p>.

Example Code

Below is an example of HTML code with explanations:

HTML Code:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example Page</title>
</head>
<body>
    <header>
        <h1>Hello, World!</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#about">About</a></li>
            <li><a href="#services">Services</a></li>
            <li><a href="#contact">Contact</a></li>
        </ul>
    </nav>
    <section id="about">
        <h2>About Us</h2>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
    </section>
    <section id="services">
        <h2>Our Services</h2>
        <p>Nulla tincidunt felis vitae odio lacinia, nec efficitur dolor viverra.</p>
    </section>
    <footer>
        <p>Copyright © 2024 Example Company. All rights reserved.</p>
    </footer>
</body>
</html>

Explanation:

  • <!DOCTYPE html>: Specifies the HTML version being used.
  • <html lang="en">: Defines the root element of the HTML document with the language attribute set to English.
  • <head>: Contains meta-information about the HTML document, such as character encoding and viewport settings.
  • <title>: Sets the title of the webpage displayed in the browser's title bar or tab.
  • <body>: Contains the visible content of the webpage, such as headings, paragraphs, and navigation.
  • <header>, <nav>, <section>, <footer>: Examples of semantic elements providing meaning to different sections of the webpage.
  • <a>: Represents a hyperlink and is used for navigation within the webpage or to external pages.
  • <ul> and <li>: Used to create an unordered list with list items, commonly used for navigation menus.
